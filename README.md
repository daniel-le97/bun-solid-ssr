# bun-solid-ssr

To install dependencies:

```bash
bun create daniel-le97/bun-solid-ssr
```



To run:
```bash
bun run dev
```

This project was created using `bun init` in bun v0.8.0. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
